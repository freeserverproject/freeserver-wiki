# 植林場について
公共の植林場と、各個人が私有する植林場とがあります。  
このページでは公共の誰でも使える植林場をリストアップします。

## 植林場を使うにあたって
どの施設でも共通の注意事項として、
- 収穫後は、苗木の植え直しをする
- アイテムを散らかしたままにしない

といったマナーを遵守しましょう。
