# 農場について
公共の農場と、各個人が私有する農地とがあります。  
このページでは公共の誰でも使える農場をリストアップします。

## 農場を使うにあたって
どの施設でも共通の注意事項として、
- 収穫後は、種の植え直しをする
- アイテムを散らかしたままにしない

といったマナーを遵守しましょう。
